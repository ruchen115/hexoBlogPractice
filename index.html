用 Hexo 輕鬆建立部落格文章

操作簡單直覺，輕鬆編寫文章

STEP 1. 在終端機下達指令 hexo new ‘文章名稱’ ，創建一篇新文章
STEP 2. 在資料夾找到並開啟 “文章名稱.md” 檔案
STEP 3. 參考 Markdown 官網 編輯文章

現在自己試試看吧!

如何用 Hexo 建立自己的網站部落格

免費架站超簡單!

STEP1. 前往 Hexo 官網，依照指示安裝 Node.js 和 Git
STEP2. 在終端機確認是否成功安裝 Node.js
STEP3. 在終端機透過 npm 下達指令安裝 hexo 套件
STEP4. 在終端機確認 hexo 套件是否安裝成功

安裝成功後在終端機輸入 hexo server，之後複製網址，前往查看自己的部落格
在終端機按 CTEL+C 即可關閉網站

現在開始可以建立自己的網站了!
